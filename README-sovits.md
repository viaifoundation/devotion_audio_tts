**GPT-SoVITS** is a powerful open-source few-shot voice cloning and TTS engine. It is ideal for cloning highly expressive voices like **Sun Wukong** (Monkey King).

## Core Features
- **Few-Shot Cloning**: Realistic results with 5-10s of audio.
- **Zero-Shot**: No training required for quick dubbing.
- **Cross-Lingual**: Speak Chinese/English/Japanese with the same voice.
